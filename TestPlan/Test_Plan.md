# Test Plan

## Project Name

Manual Testing of AutomationExercise E-Commerce Website

## Objective

The objective of this project is to perform manual testing on the AutomationExercise website to ensure that core functionalities work as expected and the application meets quality standards.

## Website Under Test

https://automationexercise.com

## Scope

The following modules will be tested:

* Homepage
* User Registration
* Login / Logout
* Product Browsing
* Product Search
* Add to Cart
* Checkout Process
* Contact Form

## Testing Types

* Functional Testing
* UI Testing
* Usability Testing
* Basic Browser Compatibility Testing

## Testing Environment

Browser: Google Chrome
Operating System: Windows 10/11

## Test Deliverables

* Test Plan
* Test Scenarios
* Test Cases
* Bug Reports
* Screenshots
